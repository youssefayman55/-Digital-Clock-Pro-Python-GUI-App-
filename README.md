# -Digital-Clock-Pro-Python-GUI-App-
A modern and interactive digital clock application built using Python and Tkinter. It displays real-time time and date with customizable themes and colors.

🚀 Project Description

This project is a GUI-based digital clock that provides:

1- 🕒 Real-time clock display

2- 📅 Live date display

3- 🌙 Dark / ☀️ Light mode toggle

4- 🎨 Custom color picker for text

5- 🖥️ Modern responsive UI

It is a beginner-friendly GUI project that demonstrates real-world Python UI development.

🛠️ Tech Stack:

1-Python 🐍

2- Tkinter (GUI framework) 🖥️

t3- ime module ⏱️

4- colorchooser 🎨

▶️ How to Run :

1- Save the script as main.py

2- Run it using:
  python main.py
  
🎯 Features :

1- 🕒 Live Digital Clock

2- Displays current time in HH:MM:SS AM/PM format

3- Updates automatically every 200ms

  Shows full date including:
  
  2- Day name

  5- 🌙 Dark / ☀️ Light Mode
  
  6- Switch between dark and light themes
